About the ISL Learning Web Application

This application is designed to help users learn Indian Sign Language (ISL) through interactive gestures and real-time feedback.

Purpose

The primary goal of this application is to provide a user-friendly interface for learning ISL, enabling users to practice and improve their signing skills.

How to Use

  1. Access the main page to start the webcam feed.
  2. Follow the target letter displayed on the screen.
  3. Make the corresponding sign using your hands.
  4. Receive immediate feedback on your performance, including your score.

Technologies Used

  • Flask - for the web framework
  • TensorFlow - for machine learning model
  • OpenCV - for webcam access and image processing
  • MediaPipe - for hand tracking

Contributors

This project is developed by Thilak and aims to promote the learning of Indian Sign Language.

About Indian Sign Language Learning Interface

This application helps you learn Indian Sign Language through an interactive interface using computer vision technology.

How It Works

The application uses your webcam to detect hand positions and gestures. Using machine learning models trained on Indian Sign Language, it can recognize the signs you make and provide instant feedback.

Instructions

  1. Position yourself in front of the camera with good lighting
  2. Make the sign for the target letter shown
  3. Hold the sign steady for the system to recognize it
  4. Get feedback on your sign accuracy
  5. Practice regularly to improve your ISL skills

Supported Signs

This application currently supports:

  • Numbers 1-9
  • Alphabet A-Z
